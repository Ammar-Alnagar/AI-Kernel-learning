# Module 04: Tile Operations

## Goal

Practice tile-local math and shape transforms:

- elementwise arithmetic
- scalar and vector broadcasting
- reshape mental model
- transpose mapping

## Exercises

1. `fused_affine`: `y = alpha * x + beta`
2. `row_bias_add`: add a 1D bias to each row
3. `reshape_2d`: verify row-major reshape logic
4. `transpose_2d`: map `(r, c)` to `(c, r)`
