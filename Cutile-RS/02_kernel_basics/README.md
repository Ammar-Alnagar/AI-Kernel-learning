# Module 02: Kernel Basics

## Goal

Practice the core launch-time and per-program indexing ideas.

- converting block id to work range
- computing per-lane offsets
- using compile-time-style constants (`BLOCK_SIZE`)

## Exercises

1. `program_range_1d`
2. `lane_offsets`
3. `copy_kernel_block`
4. `copy_launch`
