# candle_lab

Runnable crate for Candle tutorial exercises.

## Run a solution bin

```bash
./run_solution.sh module_01_tensor_fundamentals__ex01_tensor_basics
```

## Run a fill-in bin

Fill-in bins intentionally contain placeholders and will not compile until you complete them.

```bash
cargo run --bin module_01_tensor_fundamentals__ex01_tensor_basics_fill_in
```
